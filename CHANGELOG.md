# 更新记录

## 0.4.0 - 2026-07-19

- 将伤害计算与治疗计算改为独立界面模式，自动隐藏无关输入区。
- 治疗公式改为“战斗面板攻击 × 治疗倍率 ×（1 + 治疗加成）”。
- 加入基础治疗加成及觉醒、武器等候选治疗加成的生效/失效排查。
- 加入环合强度、百分比提升和固定数值提升。
- 加入仅作用于常规伤害的浸染乘区与可选特殊独立乘区。
- 创生花公式改为“9000 × 防御区 × 抗性区 ×（1 + 环合强度 / 600）”。
- 明确浸染和特殊独立乘区不作用于创生花。
- 更新 README 的游戏内字段位置、敌人抗性数据来源、公式说明和 Git 上传清单。
- 加入 MIT License。

## 0.3.0 - 2026-07-19

- 加入标准、深度和极限三档搜索强度。
- 完整枚举上限分别扩展到 18、20 和 22 个判断项。
- 完整枚举改为在线保留最优 300 个解释，避免结果数组随状态数无限增长。
- 大规模限宽分别扩展到 5,000、20,000 和 50,000 条路径。
- 搜索强度随角色和版本档案一并保存。
- 重写 README，加入实机截图、逐字段填写规则、完整结果解释和性能边界。

## 0.2.0 - 2026-07-19

- 加入可直接运行的 Windows 桌面 EXE。
- 桌面版使用固定应用数据目录，角色和版本保存不受临时解压路径影响。
- 将“最接近的效果组合”改名为“最符合实测的全局状态”。
- 将“范围内候选”改名为“±20 内来源数”，并补充验算结果解释。
- 备选全局状态只显示整批零误差结果，带误差的近似状态不再列出。
- 加入“清空当前”按钮，清除当前输入和草稿但保留已保存档案。

## 0.1.0 - 2026-07-19

- 加入常规技能、创生伤害和治疗计算。
- 支持技能倍率加算与乘算。
- 支持多个伤害技能和治疗技能。
- 加入角色与版本配置保存和草稿记忆。
- 加入基础乘区、觉醒和武器效果失效排查。
- 加入其他技能倍率误用检测。
- 加入 ±20 单项误差过滤。
- 加入完整枚举和大规模限宽搜索。
- 加入狂暴溯源模式和批量未知伤害匹配。
- 加入宽屏布局和技能、结果分页。
