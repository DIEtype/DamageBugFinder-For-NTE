# 更新记录

## 0.2.0 - 2026-07-19

- 加入可直接运行的 Windows 桌面 EXE。
- 桌面版使用固定应用数据目录，角色和版本保存不受临时解压路径影响。
- 将“最接近的效果组合”改名为“最符合实测的全局状态”。
- 将“范围内候选”改名为“±20 内来源数”，并补充验算结果解释。
- 备选全局状态只显示整批零误差结果，带误差的近似状态不再列出。
- 加入“清空当前”按钮，清除当前输入和草稿但保留已保存档案。

## 0.1.0 - 2026-07-19

- 加入常规技能、创生伤害和治疗计算。
- 支持技能倍率加算与乘算。
- 支持多个伤害技能和治疗技能。
- 加入角色与版本配置保存和草稿记忆。
- 加入基础乘区、觉醒和武器效果失效排查。
- 加入其他技能倍率误用检测。
- 加入 ±20 单项误差过滤。
- 加入完整枚举和大规模限宽搜索。
- 加入狂暴溯源模式和批量未知伤害匹配。
- 加入宽屏布局和技能、结果分页。
